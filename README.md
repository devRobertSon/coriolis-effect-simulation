# 코리올리 효과 시뮬레이션

회전하는 지구(회전 좌표계)에서 움직이는 공을 관측할 때 나타나는 **코리올리 효과**를
우주 관측자 시점과 나란히 비교해 볼 수 있는 웹 시뮬레이터입니다.

## 기능
- 두 개의 시점을 나란히 표시
  - 지구 관측자 시점 (회전 좌표계) — 궤도가 휘어져 보임
  - 우주 관측자 시점 (관성 좌표계) — 궤도는 직선
- 반구 선택: 북반구 / 남반구
- 4가지 시나리오
  1. 적도 → 극 방향
  2. 극 → 적도 방향
  3. 자전 방향 (동쪽)
  4. 자전 반대 방향 (서쪽)
- 재생 / 일시정지 / 정지
- 공 속도, 자전 속도, 시간 배율 슬라이더
- 현재 시나리오에 대한 한국어 설명 자동 표시

## 실행
정적 파일 사이트라 별도의 빌드가 필요 없습니다.

```bash
# 로컬에서 열기 (ES 모듈을 위해 정적 서버 권장)
python3 -m http.server 8000
# → http://localhost:8000
```

## GitHub Pages 배포
리포지토리의 **Settings → Pages → Branch**에서 이 브랜치(또는 `main`)의 루트(`/`)를
선택하면 `index.html`이 바로 서빙됩니다. `.nojekyll` 파일이 포함되어 있어 Jekyll
처리를 건너뜁니다.

## 기술
- [Three.js](https://threejs.org/) (CDN import map)
- Vanilla HTML / CSS / JS, 빌드 도구 없음
